# GTS Partners Database Extended Module

## Описание
Комплексный модуль для управления базой партнёров GTS Executive Panel. Предоставляет полнофункциональную систему управления агентами, подрядчиками и бренд-партнёрами с автоматизированной комиссионной системой и детальной аналитикой.

## Основные функции

### 1. Категории партнёров
**Три основных типа:**
- **Агенты**: Туристические агентства и частные агенты по продажам
- **Подрядчики**: Поставщики услуг (яхт-сервисы, вертолётные компании, гиды)
- **Бренд-партнёры**: Премиальные бренды (Mercedes-Benz Yachts, роскошные отели)

**Функциональность:**
- Фильтрация по типам партнёров через вкладки
- Унифицированные карточки с адаптивной информацией
- Быстрое добавление нового партнёра с валидацией данных
- Поиск по названию, контактному лицу или менеджеру

### 2. Карточка партнёра
**Основная информация:**
- Тип партнёрства и статус (активен/приостановлен/завершён)
- Контактные данные (название, контактное лицо, email, телефон, адрес)
- Назначенный менеджер GTS
- Дата начала сотрудничества
- Веб-сайт (для бренд-партнёров)

**Коммерческие условия:**
- Тип комиссии (процент от сделки или фиксированная ставка)
- Текущая ставка комиссии
- Период действия условий

**Статистика производительности:**
- Общее количество сделок и выручка
- Сумма заработанных и выплаченных комиссий
- Рейтинг партнёра (1-5 звёзд)
- Конверсия лидов в продажи
- Последняя активность

### 3. Комиссионная система
**Глобальные настройки:**
- Ставки по умолчанию: 10% для агентов, 20% для подрядчиков, 25% для бренд-партнёров
- Возможность индивидуальных настроек для каждого партнёра
- Переключение между процентной и фиксированной комиссией

**Автоматический расчёт:**
- Комиссии рассчитываются автоматически при подтверждении сделки
- Отслеживание статусов: начислено/подтверждено/выплачено
- Уведомления о превышении лимитов или задержках выплат

**Отчётность:**
- Сводный отчёт по всем комиссиям с разбивкой по партнёрам
- Фильтрация по периодам и типам партнёров
- Экспорт данных в CSV/Excel для внешней обработки
- Автоматические справки о доходах для партнёров

### 4. Личный кабинет партнёра (GTSPartnerPersonalCabinet)
**Обзорная панель:**
- Персональная статистика (сделки, выручка, комиссии)
- Рейтинг среди партнёров своей категории
- Достижения и награды за эффективность
- Конверсия и средний размер сделки

**Управление сделками:**
- Список всех сделок с деталями
- Статусы сделок (ожидает/подтверждено/оплачено)
- Возможность просмотра деталей каждой сделки
- Фильтрация и поиск по сделкам

**История комиссий:**
- Детальная история всех начислений
- Статусы выплат с датами
- Справки о доходах для налоговой отчётности
- Уведомления о новых начислениях

**Профиль партнёра:**
- Редактирование контактных данных (email, телефон, адрес)
- Просмотр условий партнёрского соглашения
- Скачивание документов (договор, справки)
- Связь с персональным менеджером

### 5. Аналитика партнёров
**Топ-листы и рейтинги:**
- Топ-5 партнёров по выручке с детализацией
- Рейтинг по количеству сделок и конверсии
- Сравнительный анализ эффективности по типам

**Статистика по категориям:**
- Агрегированные данные по каждому типу партнёров
- Сравнение средних показателей
- Динамика роста по категориям

**Экспорт и отчёты:**
- Генерация отчётов с настраиваемыми фильтрами
- Экспорт в популярные форматы (CSV, Excel, PDF)
- Автоматические еженедельные сводки

## Управление статусами

### Статусы партнёров
- **Активен**: Полнофункциональное сотрудничество
- **Приостановлен**: Временная приостановка (сезонность, технические причины)
- **Завершён**: Окончание партнёрства (по соглашению сторон)

### Статусы сделок
- **Ожидает**: Бронирование создано, ожидает подтверждения
- **Подтверждено**: Сделка подтверждена, комиссия зарезервирована
- **Оплачено**: Полная оплата получена, комиссия готова к выплате

### Статусы комиссий
- **Начислено**: Комиссия рассчитана, ожидает подтверждения сделки
- **Подтверждено**: Сделка завершена, комиссия готова к выплате
- **Выплачено**: Комиссия переведена партнёру

## Права доступа

### Executive и Партнёрский менеджер
- Полный доступ ко всем функциям модуля
- Создание, редактирование и деактивация партнёров
- Настройка комиссионных условий
- Просмотр всей аналитики и отчётов
- Управление выплатами комиссий

### Менеджеры отделов
- Просмотр списка партнёров своего направления
- Базовая статистика по продажам
- Без доступа к финансовым данным

### Партнёры (в личном кабинете)
- Только собственные данные и статистика
- Редактирование контактной информации
- Просмотр сделок и комиссий
- Скачивание документов и справок
