# GTS Analytics Extended Module

## Описание
Универсальный аналитический центр для GTS Executive Panel, предоставляющий руководству комплексные инструменты анализа эффективности бизнеса, AI-прогнозирование и настройки уведомлений.

## Основные функции

### 1. KPI Дашборд
- **Основные метрики**: Общее количество бронирований, выручка, доход на клиента, средний чек, загрузка техники, NPS
- **Топ-маршруты**: Лучшие маршруты по выручке с анализом роста
- **Быстрые действия**: Экспорт отчётов, уведомления, AI-ассистент
- **Фильтры**: По периоду, категории техники, менеджеру

### 2. Интерактивные графики
- **Динамика бронирований**: Линейные графики по категориям техники
- **Выручка и маржа**: Столбчатые диаграммы финансовых показателей
- **Сегментация клиентов**: Круговые диаграммы по клубным статусам
- **Эффективность персонала**: Метрики по менеджерам

### 3. Конструктор отчётов
- **Настраиваемые метрики**: Выбор KPI для анализа
- **Drill-down функциональность**: Детализация данных
- **Экспорт**: PDF, Excel форматы
- **Интерактивность**: Клик-навигация по данным

### 4. AI-Прогнозирование
- **Прогноз бронирований**: 3/6/12 месяцев с доверительными интервалами
- **Прогноз выручки**: Учёт сезонности и трендов
- **Нагрузка персонала**: Планирование штата на высокий сезон
- **AI-инсайты**: Автоматические рекомендации и предупреждения

### 5. Настройки и уведомления
- **Пороговые значения**: Настройка лимитов для KPI
- **Каналы уведомлений**: Email, Telegram, Slack, Push
- **Автоматические отчёты**: Регулярная отправка по расписанию
- **Права доступа**: Роль-ориентированный доступ к данным

## Технические особенности

### Используемые библиотеки
- **Recharts**: Для всех типов графиков и диаграмм
- **ShadCN/UI**: Унифицированные UI компоненты
- **Lucide React**: Иконки
- **React**: Hooks и state management

### Мок-данные
- Реалистичные данные для демонстрации
- Различные сценарии (рост, падение, стабильность)
- Интерактивные элементы для тестирования

### Дизайн-система GTS
- Тёмная тема портала (#0B0B0C, #121214, #17181A)
- Фирменные цвета (#91040C, #FFFFFF, #000000)
- Единые отступы и компоненты

## Роли и доступ
- **Executive**: Полный доступ ко всем данным
- **Финансовый директор**: Финансовые метрики
- **Маркетинг-менеджер**: Клиентские метрики  
- **Менеджер**: Личные показатели

## Интеграция
Модуль интегрирован в GTSExecutivePanel как вкладка "Analytics" с иконкой BarChart3 и описанием "Расширенная аналитика с AI-прогнозами".
