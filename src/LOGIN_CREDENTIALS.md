## ⚡ Быстрый доступ для тестирования

### Лендинг с аутентификацией
1. Открыть главную страницу
2. Нажать кнопку "Войти" в Header
3. Выбрать роль (member/guest/staff/partner)
4. Ввести соответствующий email
5. Пароль - любой (система в demo-режиме)

### Административные панели
1. Перейти на `/admin` или `/executive`
2. Выбрать роль через GTSUnifiedLogin
3. Ввести логин (email или телефон)
4. Пароль - любой (система в demo-режиме)

## 📖 Примечания

- **Пароль:** В текущей версии пароль не проверяется (demo режим)
- **Роли:** Автоматически назначаются на основе email
- **Права доступа:** Настроены через систему IAM с детальными разрешениями
- **Сессия:** Сохраняется в локальном state (при обновлении страницы сбрасывается)

---

## 🔄 Переключение ролей

После входа можно переключаться между ролями через:
- Profile Menu → Switch Role (в административных панелях)
- Logout → Login с другим email (на лендинге)

## 🚀 Рекомендуемый порядок тестирования

1. **Member** (`member@gts.ru`) - Протестировать лайки, комментарии, избранное, корзину
2. **Guest** (`guest@gts.ru`) - Проверить ограниченный функционал
3. **Staff** (`staff@gts.ru`) - Управление бронированиями и модерация
4. **Partner** (`partner@gts.ru`) - Работа с клиентами и комиссиями
5. **Admin** (`admin@gts.com`) - Полный административный доступ
