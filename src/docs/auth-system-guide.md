# Руководство по системе аутентификации GTS

## Обзор

Реализована полноценная система аутентификации с персонализированными страницами для разных типов пользователей Grand Tour Sochi.

## Компоненты

### 1. GTSAuthContext (`/contexts/GTSAuthContext.tsx`)

Центральный контекст аутентификации с поддержкой:
- **4 роли пользователей**: guest, member, staff, partner, executive
- **Демо-аккаунты** для тестирования
- **Управление сессией**: login, logout, updateUser
- **Данные пользователя**: имя, email, роль, аватар, уровень членства, баллы, разрешения

### 2. Демо-аккаунты

```typescript
// Член клуба (Gold Member)
email: member@gts.ru
permissions: комментировать, лайкать, бронировать, избранное

// Гость 
email: guest@gts.ru
permissions: просмотр, бронирование, избранное

// Сотрудник
email: staff@gts.ru
permissions: управление бронированиями, отчеты, модерация

// Партнер
email: partner@gts.ru
permissions: создание бронирований, комиссии, клиенты
```

*Любой пароль подойдет для демо*

## Авторизованные страницы

### GTSAuthExperiencesPage

**Особенности для авторизованных пользователей:**

✅ **Персональная панель статуса**
- Аватар и имя пользователя
- Уровень членства (Silver/Gold/Platinum)
- Количество баллов
- Счетчик избранного и корзины
- Быстрый переход в личный кабинет

✅ **Система избранного (Favorites)**
- Кнопка-сердечко на каждой карточке
- Визуальная индикация (заливка красным)
- Сохранение в локальном состоянии

✅ **Корзина (Shopping Cart)**
- Добавление впечатлений в корзину
- Кнопка меняется: "В корзину" → "В корзине" (с галочкой)
- Счетчик в шапке профиля
- Кнопка "Оформить (N)" для перехода к оформлению

✅ **Персональные рекомендации**
- Для Gold/Platinum members
- Специальные скидки -15%
- Крупные карточки с градиентом
- Отображаются вверху страницы

✅ **Визуальные улучшения**
- Бейджи со скидками
- Рейтинги и отзывы
- Цены со скидками (перечеркнутая старая цена)

### GTSAuthStoriesPage

**Особенности для авторизованных пользователей:**

✅ **Публикация историй**
- Кнопка "Поделиться историей"
- Форма с заголовком и текстом
- Кнопки добавления фото/видео/локации
- Доступно для members с разрешением "comment"

✅ **Лайки**
- Кнопка-сердечко под каждой историей
- Счетчик лайков
- Визуальная индикация (красный цвет при лайке)
- Анимация при клике

✅ **Комментарии**
- Раскрываемый блок комментариев
- Показ существующих комментариев
- Форма добавления нового комментария
- Аватар автора комментария
- Отправка по Enter или кнопке

✅ **Сохранение в избранное**
- Кнопка-закладка на изображении
- Визуальная индикация (заливка при сохранении)

✅ **Фильтрация**
- По категориям: все, морские, приключения, premium, сообщество
- Сохранение фильтра при навигации

## Навигация

### GTSNavigationHeader

**Для неавторизованных:**
- Кнопка "Войти"

**Для авторизованных:**
- Аватар пользователя
- Имя и уровень членства
- Выпадающее меню:
  - Личный кабинет
  - Баллы лояльности
  - Выход

### GTSRouter

Автоматически показывает:
- `GTSAuthExperiencesPage` вместо `GTSExperiencesPage` для авторизованных
- `GTSAuthStoriesPage` вместо `GTSStoriesPage` для авторизованных

## Страница входа (GTSLoginPage)

### Процесс входа:

1. **Выбор роли:**
   - Член клуба
   - Сотрудник  
   - Партнер

2. **Ввод данных:**
   - Email
   - Пароль
   - Чекбокс "Запомнить меня"

3. **Демо-подсказки:**
   - Для роли "Член клуба" показываются доступные демо-аккаунты
   - Подсказка что любой пароль подойдет

4. **После входа:**
   - Member → страница впечатлений (авторизованная версия)
   - Employee → executive panel
   - Partner → partner portal

## Разрешения (Permissions)

### Guest
- `view-content` - просмотр контента
- `book-experiences` - бронирование
- `save-favorites` - избранное

### Member  
- Все из Guest +
- `comment` - комментирование
- `like` - лайки
- `view-my-bookings` - свои бронирования

### Staff
- `view-content`
- `manage-bookings` - управление бронированиями
- `view-reports` - отчеты
- `moderate-comments` - модерация

### Partner
- `view-content`
- `create-bookings` - создание бронирований
- `view-commissions` - просмотр комиссий
- `manage-clients` - управление клиентами

## Интеграция

```tsx
// В App.tsx
<GTSAuthProvider>
  <GTSRouter />
</GTSAuthProvider>

// Использование в компонентах
import { useGTSAuth } from '../contexts/GTSAuthContext';

function MyComponent() {
  const { user, isAuthenticated, login, logout } = useGTSAuth();
  
  return (
    <div>
      {isAuthenticated ? (
        <p>Привет, {user.name}!</p>
      ) : (
        <button onClick={() => login('member@gts.ru', 'any')}>
          Войти
        </button>
      )}
    </div>
  );
}
```
